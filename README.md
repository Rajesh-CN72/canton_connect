# Canton Connect - Flutter Food Delivery App

A modern, responsive Flutter application for food delivery with customer and admin interfaces.

## Features

### Customer App
- Browse restaurant menu with categories
- Add items to cart with customization
- Place orders with real-time tracking
- Manage subscription plans
- Multi-language support (English/Chinese)
- User profile and order history

### Admin Dashboard
- Manage food items and menu
- Process and track orders
- Customer management
- Subscription management
- Analytics and reports

### Technical Features
- Responsive design (mobile, tablet, desktop)
- Provider state management
- GoRouter for navigation
- REST API integration
- Push notifications
- Payment gateway integration
