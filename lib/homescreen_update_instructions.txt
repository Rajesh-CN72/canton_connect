HOW TO UPDATE YOUR HOMESCREEN:

1. IMPORT THE NEW WIDGETS:
import '../widgets/category_grid.dart';
import '../widgets/featured_chefs_grid.dart';
import '../widgets/smart_carousel.dart';

2. REPLACE EXISTING SECTIONS:

// Replace categories section with:
CategoryGrid(
  onCategoryTap: (categoryId) {
    // Handle category tap
    print('Category tapped: \');
  },
),

// Replace chefs section with:
FeaturedChefsGrid(
  onChefTap: (chefId) {
    // Handle chef tap
    print('Chef tapped: \');
  },
),

// Replace carousel section with:
SmartCarousel(
  onSectionTap: (sectionId) {
    // Handle section tap
    print('Section tapped: \');
  },
),

3. UPDATE CUSTOM APP BAR:
CustomAppBar(
  title: 'Home',
  showBackButton: false,
),
